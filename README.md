# Aplicación ToDo Flask 

Aplicación web de lista de tareas construida con **Flask**, **SQLite** y **SQLAlchemy**. Permite agregar y visualizar tareas en una interfaz sencilla.

---

## Tecnologías

- Python 3.12
- Flask
- SQLAlchemy
- HTML5 + CSS3
